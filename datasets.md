### 1. Open CEDA 2025 (Spend-based EEIO – Scope 3 fallback)

- Main access page: **[https://openceda.org](https://openceda.org)** (fill the form to get the download link via email)
- AWS Open Data (direct S3 bucket): **[Open CEDA by Watershed - Registry of Open Data on AWS](https://registry.opendata.aws/open-ceda/)** → You can browse/explore files here: s3://open-ceda/ (use AWS CLI with --no-sign-request)

### 2. DEFRA 2025 GHG Conversion Factors (Activity-based – best for Scope 1 & 3)

- Official page: **[GOV.UK](https://www.gov.uk/government/publications/greenhouse-gas-reporting-conversion-factors-2025)**
- **Direct full set download** (recommended): **https://assets.publishing.service.gov.uk/media/6846a4f55e92539572806125/ghg-conversion-factors-2025-full-set.xlsx**

### 3. EEA Electricity Emission Intensity (Scope 2 – EU27 country-specific)

- Main indicator page (with latest 2024 data published 2025): **https://www.eea.europa.eu/en/analysis/indicators/greenhouse-gas-emission-intensity-of-1**
- You can download country-level tables/CSV directly from the charts or data section on this page.

### 4. EDGAR 2025 (JRC/EC) – Validation & National Baselines

- Main report & downloads page: **[EDGAR - The Emissions Database for Global Atmospheric Research](https://edgar.jrc.ec.europa.eu/report_2025)**
- Direct Excel downloads:
  - EDGAR GHG emissions: **[EDGAR_2025_GHG_booklet_2025.xlsx](https://edgar.jrc.ec.europa.eu/booklet/EDGAR_2025_GHG_booklet_2025.xlsx)**
  - Full booklet (PDF): Available on the same page.

### 5. EXIOBASE 3 (Deeper MRIO for Scope 3)

- Latest version (3.9.4 as of early 2025): **https://zenodo.org/records/14614930**
- Alternative / previous stable: Search Zenodo for “EXIOBASE 3” for newer uploads if available.

### 6. IPCC Emission Factor Database (EFDB) – Scope 1

- Main searchable database: **https://www.ipcc-nggip.iges.or.jp/EFDB/main.php**
- Use the **Basic Search** or **Fulltext Search** and export results to Excel. (No single bulk file — best to search by sector/gas and export as needed.)
